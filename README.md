# MS Dhoni - CSK Legend

A modern, responsive web page dedicated to MS Dhoni, celebrating his legendary cricket career with Chennai Super Kings and the Indian national team.

## 🏆 Overview

This project showcases the remarkable journey of MS Dhoni, also known as "Captain Cool," one of cricket's greatest leaders. The website features information about his leadership, achievements, and legacy in both international cricket and the Indian Premier League (IPL).

## ✨ Features

### 1. **Responsive Navigation**
   - Clean, modern header with navigation menu
   - Hamburger menu for mobile devices
   - Active link highlighting based on scroll position
   - Smooth scroll behavior on all navigation links

### 2. **Hero Section**
   - Eye-catching hero banner with call-to-action button
   - Engaging headline and description
   - Links to explore more content

### 3. **Features Section**
   - Three highlight cards showcasing Dhoni's key attributes:
     - **Leadership**: Multiple IPL titles under his captaincy
     - **Finishing Skills**: Known for calm demeanor and match-winning finishes
     - **Jersey Number 7**: The legendary and iconic number in cricket

### 4. **About Section**
   - Comprehensive biography covering:
     - Early life and rise to stardom
     - Captaincy achievements (T20 World Cup 2007)
     - IPL success with Chennai Super Kings
     - Philanthropic efforts and interests
     - Major awards and accolades

### 5. **Contact Section**
   - Contact information area for user inquiries

### 6. **Sticky Footer**
   - Copyright information
   - Professional footer styling

## 🛠️ Technology Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling and animations (from `styles.css`)
- **JavaScript**: Interactive features and smooth navigation
- **Font Awesome 6.0.0**: Icon library for enhanced UI

## 📱 Responsive Design

The website is fully responsive and works seamlessly across:
- Desktop computers
- Tablets
- Mobile devices (with hamburger menu for smaller screens)

## 🎯 Key JavaScript Features

1. **Smooth Scrolling**: Navigate smoothly to different sections
2. **Dynamic Header**: Header styling changes on scroll
3. **Active Link Detection**: Navigation links highlight based on viewport position
4. **Mobile Menu Toggle**: Hamburger menu for mobile navigation

## 🚀 Getting Started

### Prerequisites
- No build tools or dependencies required
- Works with any modern web browser

### Installation

1. Clone or download the repository:
```bash
git clone https://github.com/Adithya3633/PRODIGY_WD_1.git
```

2. Navigate to the project directory:
```bash
cd PRODIGY_WD_1
```

3. Open `index.html` in your web browser:
   - Simply double-click the file, or
   - Use a local server for better experience

### Running Locally (with a local server)

Using Python:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Then visit: `http://localhost:8000`
